SVGarden

SVGarden is a React and Vite web application that allows users to browse and download SVG files organized into categories. The website was designed as a creative resource for Cricut Design Space projects and crafting inspiration.

Users can explore different categories, preview SVG designs, and view organized image galleries in a clean and responsive layout.

Features: Category-based SVG organization, Responsive image gallery, Category preview cards, Downloadable SVG assets, React Router navigation, Dynamic filtering by category, and Clean and simple user interface.
